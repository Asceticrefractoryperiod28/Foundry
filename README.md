If you want to build a content agency, state that clearly. If you want to build a software testing service, list the specific goals for your agents.

### Organizing Your Agents
Foundry assigns specialized agents based on your mission. Each agent has a title and a set of responsibilities. You can view the list of active agents in the sidebar. You can add new agents if you need to tackle different parts of your project.

### Starting the Workflow
Click the Start button on your dashboard. The agents inside the system will begin their work. You can watch their progress in the activity feed. ## ❓ Frequently Asked Questions

**Does this require permanent internet access?**
The software needs internet for the AI components to function. It cannot process the complex logic required for the agents without cloud connectivity.

**Can I run multiple companies at once?**
Yes. You can switch between different company dashboards using the menu on the left side of the window.

**What happens if the agents stop working?**
Check your internet connection first. If the network is stable, click the refresh button on your dashboard. This forces the agents to re-sync their current tasks.

**Does this cost money to use?**
The software is free to download. Some advanced AI models that you choose to connect might charge fees based on usage. That depends on the specific provider you choose within the settings.

**Can I see what the agents write?**
Yes. Every output appears in the activity log. You can copy this text to use in other documents or save it as a local file.
